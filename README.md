# Solomon Marvelous - Website

Personal landing site for Solomon Marvelous (Enterprise Architect & technology leader), built with Vue 3, TypeScript, and Vite.

## Requirements

- [Node.js](https://nodejs.org/) (LTS recommended)

## Setup

```bash
npm install
```

## Scripts

| Command        | Description                    |
| -------------- | ------------------------------ |
| `npm run dev`  | Start dev server (Vite)        |
| `npm run build`| Type-check and production build |
| `npm run preview` | Serve the `dist` output locally |

## Stack

Vue 3, Vue Router, Vite, TypeScript, Tailwind CSS, GSAP, Radix Vue, Phosphor Icons, Video.js.
